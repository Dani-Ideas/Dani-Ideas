## Main project — live in production

### ERP / POS — Tech retail system
> Next.js 16 · TypeScript · PostgreSQL · Prisma ORM · NextAuth v5 · Stripe · Mercado Pago · Supabase · Vercel

Built solo from scratch. **Currently used by real clients.**

- Full inventory module: Kardex (StockMove), batches, hierarchical locations, transfers, receipts
- Real payment processing: Stripe checkout + Mercado Pago Point with split-payment in POS
- NextAuth v5 auth, cash register session control, complete audit trail
- QR per physical unit with auto inventory relocation on scan
- XLSX export · Atomic DB transactions · Continuous deploy on Vercel

## Job Scraper Dashboard

> Python · Playwright · asyncio · Next.js 14 · TypeScript · Tailwind CSS
